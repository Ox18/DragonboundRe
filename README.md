# 1. Server Basic

### Utilities:
- [DataBase.sql](https://github.com/alexwathever/Openbound-HTML5/blob/main/Client%20-%20Basic%20-%20v1-119/DataBase/dragonbound.sql)

### Requirements:
- NodeJS (v6.8)
- MySQL

### Install Modules:
```
npm install
```

### Run server:
```
node web.js
node game.js
```
